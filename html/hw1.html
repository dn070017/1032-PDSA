Assignment

Poker Game:
In the Chinese New Year, cognates play poker game in family parties. Each player is dealt to five cards. Your program is requested to identify which players have a hand of Flush (A flush is a poker hand, where all five cards are of the same suit).

In this homework, there are three Java classes involved. Please design the Game (main) Class and implement one function of the Player Class. The Game Class deals five cards to each player (the number of players is given in the first line of the input file), and calls the function of the Player Class to judge if it is a hand of Flush. In the end, print out the players who have a hand of Flush.

Files

Game.java, Player.java, Card.java (do not modify 'Card.java')

Input Example:hw1.in.txt
4
Kristoff
Hearts_10,Hearts_9,Hearts_Q,Hearts_5,Hearts_3
Elsa
Spades_A,Hearts_A,Hearts_4,Diamonds_4,Clubs_4
Anna
Diamonds_J,Diamonds_K,Diamonds_2,Diamonds_8,Diamonds_9
Olaf
Clubs_A,Hearts_8,Diamonds_Q,Diamonds_10,Clubs_2

Output:
Kristoff
Anna

Submission

When you are doing the homework, you put all the three class files in the same project. When submitting the homework, you will be requested to upload 'Player.java' and 'Game.java' separately. These two files will be judged independently, which means that you are not allowed to modify the interface of the functions.

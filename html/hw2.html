Assignment 2-2: Read Pre-grouping for Assembly

Next-generation sequencing (NGS) is a newly developed technology that has the power to deliver short DNA reads with a considerably low cost. One important task in DNA sequencing is to assemble these short reads into longer DNA sequences. This homework assignment is not going to assemble these short reads into final sequences. Instead, you are asked to perform a task called pre-grouping of reads, which can later be used by assemblers.

In this assignment, the input would be a lot of k-mer reads and you should group reads into connected components based on the overlap principal. The principal requires the overlap of two reads larger than or equal to n (k = length of reads, n = minimal overlap length). For examples, if k = 10 and n = 3, and if the overlap between two reads is larger than or equal to 3, these two reads are considered related and should be put in the same group. In other words, overlap lengths 10, 9, 8, 7, 6, 5, 4, 3 are qualified.

A small example of read set

read1:__________AGAGCTAGCA
read2:___AGCTGCTAGA
read3:___________GATTATACTG
read4:_TCAGCTGCTA
In the configuration of k = 10 and n =3, read1 and read2 must be grouped together (overlap = 3), but read2 and read3 are not (overlap = 2). In the end, read1, read2, and read4 are in the same group because of the qualified overlap between read1 and read2 and the qualified overlap between read2 and read4.

In HW2-2, you are requested to write a program named ReadAssembler.java, which finally outputs the number of groups (connected components). For this small example, your program should output '2'.

Sample input hw2-2.in.txt

Sample output
3

Note:
Your program should check whether all the reads in the input file are with a length of k. If not, your program should terminate properly and print "Input error!" to the standard output.
